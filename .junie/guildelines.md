Это бэкенд для телеграм бота. Это основной сервис всей системы. Помимо этого сервиса есть еще:
1. Сервис отвечающий за рассылку/отправку сообщений
2. Сервис формирования пуш сообщений
3. Сервис приема данных от телеграм

Сервисы в системе обмениваются данными посредством сервера рэбитMQ и https запросов.  Данный сервис работает на yii3, используй возможности этого фреймворка. Пиши простой, понятный, читаемый код.

Данный сервис отвечает за обработку данных от телеграм. Производите логические операции, работает с базами данных и отправляет ответы через сервис отправки сообщений.
Данные он получает из rabbitMQ, может быть запущенно сразу несколько экземпляров этого приложения.

В RabbitMQ у нас есть несколько очередей
1. Очередь с полученными данными от телеграм
2. Очередь с отправляемыми сообщениями
